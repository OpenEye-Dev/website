The Open Eye Platform

We've released an open source API for Diabetic Retinopathy Grading. We use the latest in deep learning and scalable web technologies to deliver an open platform that can automatically grade fundus images for DR severity.

Try the Demo

Simple Design

We aim to build the simplest end-to-end solution for Diabetic Retinopathy Screening, from the hardware needed for imaging, to the software that provides automatic analysis.

Open Source

All source code and designs are released using an open source license. You can immediately download our solution and build it yourself. You can also contribute to the project and make it better.

Easy to Use

We've built this solution to be used by an untrained end-user, making it ideal for screening scenarios where experts are not available. Our community is dedicated to making it easy to implement our solution.
